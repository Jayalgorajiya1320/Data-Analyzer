📊 Data Analytics & Excel Dashboard Project
📌 Project Overview

This project demonstrates data analysis using Microsoft Excel, focusing on cleaning data, applying formulas, building KPIs, and creating an interactive dashboard. It is designed for learning and portfolio purposes and is suitable for beginners to intermediate data analysts.

The project covers:

    Sales analysis
    Customer insights
    Product and region-wise performance
    Visual dashboards for decision-making

🛠 Tools & Technologies

    Microsoft Excel
    Excel Functions: SUMIFS, COUNTIFS, INDEX, MATCH, IF, VLOOKUP
    Pivot Tables & Pivot Charts
    Data Validation & Named Ranges
    Conditional Formatting

📂 Dataset Description

The dataset contains sales transaction data with the following columns:
Column Name 	Description
Customer_ID 	Unique customer identifier
Customer_Name 	Name of the customer
Region 	Sales region (East, West, North, South)
Product_Category 	Product type
Sales 	Total sales amount
Quantity 	Units sold
Discount 	Discount applied
Order_Date 	Date of order
Profit 	Profit earned
📈 Key Analysis Performed
1️⃣ KPIs

    Total Sales
    Total Profit
    Total Quantity Sold
    Average Discount

2️⃣ Monthly Sales Trend

    Line chart to visualize month-wise sales growth

3️⃣ Sales by Product Category

    Pie chart showing contribution of each product category

4️⃣ Region-wise Performance

    Bar chart comparing sales across regions

5️⃣ High-Value Customers

    Identified using INDEX + MATCH and filtering techniques

📊 Dashboard Features

    Interactive Pivot Charts
    Clean and professional layout
    Easy-to-understand KPIs
    Dynamic filtering using slicers
